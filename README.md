# Frontend Next.js Application

This is a frontend web application built using React and Next.js. It serves as the user interface layer and follows modern best practices for performance, scalability, and maintainability.

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules, Tailwind CSS
- Node.js
- ESLint and Prettier

## Prerequisites

Make sure you have the following installed:

- Node.js v18 or later
- npm, yarn, or pnpm

Verify installation:

node -v  
npm -v  

## Installation

Clone the repository:

git clone https://github.com/neuroacura-git/neuroacura-web
cd neuroacura-web

Install dependencies:

npm install  
or  
yarn install  
or  
pnpm install  

## Running the Development Server

Start the development server:

npm run dev  
or  
yarn dev  
or  
pnpm dev  

Open your browser and navigate to:

http://localhost:3000

The application will automatically reload on code changes.

## Build for Production

Create a production build:

npm run build  

Start the production server:

npm run start  

## Environment Variables

Create a .env.local file in the project root:

check out .env.sample file for the keys you need to input

Do not commit environment files to version control.

## Scripts

npm run dev     Start development server  
npm run build   Build for production  
npm run start   Start production server  
npm run lint    Run linting
